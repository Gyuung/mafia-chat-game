## 2026-05-28

### 🛠 주요 작업 내용

1.  **React Doctor 전수 진단 및 최적화 (점수: 100/100 달성)**
    *   **컴포넌트 파편화**: 434라인의 `MafiaGame.tsx`를 분석하여 `sub/` 디렉토리로 8개의 컴포넌트를 추출. 이 과정에서 `Props Drilling`을 방지하기 위한 인터페이스 재설계 수행.
    *   **접근성 보완**: 모든 인터랙티브 요소(버튼, 입력창)에 `aria-label` 및 `aria-pressed` 속성 추가.
    *   **수화(Hydration) 이슈 해결**: `new Date()`를 렌더링 시점에 직접 호출하지 않고 `useEffect`나 `useMemo`를 통해 클라이언트 사이드에서만 결정되도록 수정하여 서버 렌더링과의 불일치 제거.

2.  **프로젝트 인프라 정비**
    *   `react-doctor` 구동 환경에 맞춰 Node.js 버전을 `v22.22.3`으로 강제하고 `.nvmrc`에 기록.
    *   과거 유실된 문서(`CHANGELOG_INTERNAL.md`, `AGENT_WORK_LOG.md`)를 Git 히스토리를 대조하며 전수 복원.
