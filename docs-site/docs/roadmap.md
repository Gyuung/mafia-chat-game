# 로드맵

## 완료된 항목

- XP, 레벨, 칭호, 플레이 기록 `localStorage` 저장 및 누적 시스템
- 상세 결과 카드 및 투표 요약 섹션 구현 (역할 아이콘 및 레이아웃 개선)
- 하루 1회 보너스를 제공하는 '오늘의 사건' 모드 (고난도 모드 포함)
- 채팅 플랫폼 응답 미리보기 (명령 예시, 버튼 선택지, 축약 결과)
- 가상 참가자의 대화 다양성 및 상황별 발언 확장
- 가상 참가자 간의 '신뢰/호감도' 시스템 및 투표 로직 반영

## 다음 구현 후보

- 게임 결과 로그 내보내기 기능 (텍스트 또는 이미지)
- 가상 참가자의 이름 및 성격 유형 다양화 (소심함, 공격적, 논리적 등)
- 게임 설정 화면 UI 개선 (일반 모드 난이도 조절 등)
- 주요 사건 발생 시 사운드 효과 또는 시각적 피드백 추가

## 나중에 검토할 것

- 실제 멀티 참가자 모드
- 서버 저장소와 계정 연동
- 운영자용 밸런스 설정
- 대화 템플릿 관리 화면
- 배포 환경별 문서 URL 분리
