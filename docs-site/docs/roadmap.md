# 로드맵

## 완료된 항목

- XP, 레벨, 칭호, 플레이 기록 `localStorage` 저장 및 누적 시스템
- 상세 결과 카드 및 투표 요약 섹션 구현
- 하루 1회 보너스를 제공하는 '오늘의 사건' 모드 및 브리핑 시스템
- 채팅 플랫폼 응답 미리보기 (명령 예시, 버튼 선택지, 축약 결과)
- 가상 참가자의 투표 근거 로그 표시 및 의심도 기반 투표 로직

## 다음 구현 후보

- 가상 참가자의 대화 다양성 및 상황별 발언 확장
- 참가자 간의 '신뢰/호감도' 시스템 (우호적인 플레이어 보호 등)
- 투표 요약 화면의 시각적 개선 (역할 아이콘, 레이아웃 등)
- 더 공격적인 NPC가 등장하는 '오늘의 사건' 고난도 모드

## 나중에 검토할 것

- 실제 멀티 참가자 모드
- 서버 저장소와 계정 연동
- 운영자용 밸런스 설정
- 대화 템플릿 관리 화면
- 배포 환경별 문서 URL 분리
