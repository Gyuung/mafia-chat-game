# Mafia Chat Game Docs

Mafia Chat Game은 혼자 접속해도 가상 참가자들과 진행할 수 있는 채팅형 마피아 게임입니다. 현재 목표는 웹에서 바로 플레이 가능한 프로토타입을 안정화하고, 이후 채팅 플랫폼에서도 같은 규칙과 결과 표현을 재사용할 수 있게 구조를 정리하는 것입니다.

## 현재 방향

- 첫 화면은 랜딩 페이지가 아니라 바로 플레이 가능한 게임 화면입니다.
- 플레이어는 자기 역할만 알고 시작합니다.
- 가상 참가자의 역할은 게임 종료 후 공개됩니다.
- 밤 행동, 낮 심문, 투표, 승패 판정, XP 보상을 한 흐름으로 관리합니다.
- 규칙, 변경 이력, 다음 작업은 이 문서 사이트에서 버전 관리합니다.

## 실행

루트 앱:

```bash
npm run dev
```

문서 사이트:

```bash
npm run docs:dev
```

로컬 기준 게임은 `http://localhost:3000`, 문서는 `http://localhost:3001`에서 확인합니다.
