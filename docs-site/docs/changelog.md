# 변경 이력

## 0.8.0 - 2026-05-25

- **추리 시스템 심화 및 게임 가이드 추가**:
  - **언어적 단서(Tells)**: 이제 마피아 NPC는 심문 시 말을 번복하거나, 대화 중 지나치게 방어적인 태도를 보이는 등 식별 가능한 단서를 노출합니다.
  - **심문 일관성**: 시민은 항상 같은 알리바이를 말하지만, 마피아는 확률적으로 진술이 꼬일 수 있습니다.
  - **게임 가이드 도입**: 대기 화면에 마피아를 찾는 방법과 플레이 팁을 정리한 가이드 섹션을 추가하여 초보자도 쉽게 추리를 시작할 수 있도록 돕습니다.

## 0.7.3 - 2026-05-25

- **모바일 최적화 및 레이아웃 개선**: 
  - 스마트폰 및 태블릿 환경에 맞춰 사이드바를 접이식 메뉴로 변경하고, 상단 바를 통해 게임 상태를 한눈에 볼 수 있도록 UI를 조정했습니다.
  - 버튼과 입력창의 크기를 모바일 터치에 최적화하여 조작 편의성을 높였습니다.

## 0.7.2 - 2026-05-25

- **NPC 대화 피드백 시스템 도입**: 채팅 로그의 NPC 메시지에 마우스를 올리면 나타나는 '좋아요/싫어요' 버튼을 통해 대사 품질을 평가할 수 있는 기능을 추가했습니다. 수집된 피드백은 향후 대사 로직 개선에 활용됩니다.

## 0.7.1 - 2026-05-25

- **탈락 연출 강화**: 참가자가 탈락할 때(밤 공격 또는 투표) 화면이 흔들리고 붉은 선혈 효과가 나타나는 시각적 피드백을 추가하여 긴장감을 극대화했습니다.

## 0.7.0 - 2026-05-25

- **시각적 애니메이션 및 몰입도 강화**: 
  - 채팅 메시지가 부드럽게 등장하는 애니메이션을 추가했습니다.
  - 단계 전환 오버레이를 더 화려하고 직관적으로 개선했습니다. (밤 단계 시 붉은색 펄스 효과 추가)
  - 결과 카드 등장 시 페이드 인 효과를 적용했습니다.
- **마피아 역할 지표 수정**: 마피아 승리 시 '밤 공격 성공' 횟수가 올바르게 표시되도록 수정했습니다.

## 0.6.9 - 2026-05-25

- **결과 카드 상세 지표 강화**: 게임 종료 시 단순 승패 외에도 '심문 횟수', '정확한 투표 횟수', '역할별 특수 능력 성공 횟수(의사 방어, 경찰 조사 등)'를 확인할 수 있는 상세 지표를 추가했습니다.
- **결과 공유 내용 확장**: 클립보드 복사 시 추가된 상세 지표들이 함께 포함되도록 개선했습니다.

## 0.6.8 - 2026-05-25

- **NPC 대화 컨텍스트 인지 강화**: NPC가 대화 중 다른 참가자의 이름을 언급하거나, 마피아 검거 후 상황에 맞는 승리 분위기/경계 대사를 하도록 개선하여 몰입도를 높였습니다.

## 0.6.7 - 2026-05-25

- **결과 공유 기능 추가**: 게임 종료 후 결과 카드에서 '공유하기' 버튼을 통해 전체 게임 요약(결과, 역할, XP, 주요 사건 등)을 클립보드에 복사할 수 있는 기능을 추가했습니다.

## 0.6.6 - 2026-05-25

- **NPC 대화 다양성 강화**: 밤 사이 탈락자가 발생하거나 게임 후반부 진입 시 상황에 맞는 대사를 하도록 개선했습니다.
- **결과 카드 상세 지표 추가**: 게임 종료 시 '마피아 검거 수'와 '총 진행 라운드' 정보를 보여줍니다.

## 0.6.5 - 2026-05-25

- **로그 내보내기 기능 추가**: 게임 진행 로그를 .txt 파일로 저장할 수 있는 기능을 추가했습니다.
- **NPC 성격 및 이름 다양화**: '냉소파' 성격을 추가하고, 가상 참가자들의 이름을 25종으로 확장했습니다.
- **게임 난이도 설정 도입**: 일반 게임 시작 시 '쉬움', '보통', '어려움' 난이도를 선택할 수 있습니다. 난이도에 따라 NPC의 투표 성향이 달라집니다.
- **단계 전환 연출 추가**: 밤과 낮이 바뀔 때 화면에 단계 이름을 보여주는 시각적 피드백을 추가했습니다.

## 0.6.4 - 2026-05-25

- **NPC 성격 시스템 도입**: 가상 참가자들에게 논리파, 공격파, 신중파, 감정파 4가지 성격을 부여하여 말투와 투표 성향을 다양화했습니다.
- **참가자 목록 개선**: 사이드바의 참가자 목록에서 각 NPC의 성향(예: 논리파)을 확인할 수 있는 기능을 추가했습니다.

## 0.6.3 - 2026-05-24

- 오늘의 사건 종류를 4종에서 7종으로 확장하고 보상 XP 밸런스를 조정했습니다.
- 가상 참가자의 투표 로직을 실제 역할 정보 대신 의심도(Suspicion) 기반으로 더 정교하게 튜닝했습니다.
- 결과 카드에 누가 누구에게 투표했는지 보여주는 '투표 요약' 섹션을 추가했습니다.

## 0.6.2 - 2026-05-23

- 가상 참가자의 투표 선택 이유를 진행 로그에 남겨 의사결정 근거를 확인할 수 있게 했습니다.

## 0.6.1 - 2026-05-22

- 게임 시작 전 화면에 명령 예시, 버튼 선택지, 축약 결과 카드 형식을 보여주는 채팅 응답 미리보기 패널을 추가했습니다.
- 날짜별 브리핑과 하루 1회 보너스 XP를 제공하는 오늘의 사건 모드를 추가했습니다.
- 결과 카드에 팀 결과, 획득 XP, 레벨 진행도, 칭호 변화, 주요 사건 요약을 추가했습니다.
- 가상 참가자의 랜덤 이름을 실제 사람 이름 대신 익명 닉네임형 이름으로 변경했습니다.

## 0.6.0 - 2026-05-21

- XP, 레벨, 칭호, 최근 플레이 기록을 브라우저 `localStorage`에 저장하도록 변경했습니다.
- 새 게임이나 게임 리셋 후에도 누적 XP와 레벨 진행도가 유지됩니다.
- 사이드바에 누적 XP, 현재 칭호, 최근 플레이 기록 요약을 표시했습니다.

## 0.5.0 - 2026-05-18

- Docusaurus 기반 문서 사이트를 `docs-site/`로 분리했습니다.
- 루트 스크립트에 `docs:dev`, `docs:build`, `verify:all`을 추가했습니다.
- 게임 화면의 Docs 링크를 별도 문서 사이트로 연결했습니다.
- 템플릿 샘플 문서, 블로그, 기본 이미지를 제거했습니다.

## 0.4.0 - 2026-05-18

- 낮 심문 기능을 추가했습니다.
- 역할별 카드 이미지를 추가했습니다.
- 진행 로그에 새 메시지가 생기면 자동으로 아래로 스크롤되도록 했습니다.
- 결과 카드에 XP, 레벨, 칭호 정보를 추가했습니다.

## 0.3.0 - 2026-05-18

- 혼자 플레이하는 가상 참가자 구조를 추가했습니다.
- 내 역할만 공개하고, 다른 역할은 게임 종료 후 공개하도록 바꿨습니다.
- 가상 참가자의 자동 발언과 투표 로직을 추가했습니다.

## 0.2.0 - 2026-05-18

- 라운드 진행, 밤 행동, 낮 토론, 투표, 승패 판정을 구현했습니다.

## 0.1.0 - 2026-05-18

- Next.js 기반 초기 프로젝트를 만들었습니다.
- 자동 커밋 스크립트와 기본 프로젝트 문서를 추가했습니다.
