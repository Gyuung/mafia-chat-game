# 변경 이력

## 0.6.6 - 2026-05-25

- **NPC 대화 다양성 강화**: 밤 사이 탈락자가 발생하거나 게임 후반부 진입 시 상황에 맞는 대사를 하도록 개선했습니다.
- **결과 카드 상세 지표 추가**: 게임 종료 시 '마피아 검거 수'와 '총 진행 라운드' 정보를 보여줍니다.
- **CLI 사용성 개선**: 쉘 명령어 실행 시 매번 묻는 확인 절차를 생략하도록 자동 허용 정책을 설정했습니다.

## 0.6.5 - 2026-05-25

- **로그 내보내기 기능 추가**: 게임 진행 로그를 .txt 파일로 저장할 수 있는 기능을 추가했습니다.
- **NPC 성격 및 이름 다양화**: '냉소파' 성격을 추가하고, 가상 참가자들의 이름을 25종으로 확장했습니다.
- **게임 난이도 설정 도입**: 일반 게임 시작 시 '쉬움', '보통', '어려움' 난이도를 선택할 수 있습니다. 난이도에 따라 NPC의 투표 성향이 달라집니다.
- **단계 전환 연출 추가**: 밤과 낮이 바뀔 때 화면에 단계 이름을 보여주는 시각적 피드백을 추가했습니다.

## 0.6.4 - 2026-05-25

- **NPC 성격 시스템 도입**: 가상 참가자들에게 논리파, 공격파, 신중파, 감정파 4가지 성격을 부여하여 말투와 투표 성향을 다양화했습니다.
- **코드 구조 대규모 리팩토링**: 거대했던 `MafiaGame.tsx` 파일을 기능별 모듈(types, constants, hooks, sub-components)로 분리하여 유지보수성을 높였습니다.
- **참가자 목록 개선**: 사이드바의 참가자 목록에서 각 NPC의 성향(예: 논리파)을 확인할 수 있는 기능을 추가했습니다.
- **AI 커밋 스크립트 강화**: 다양한 AI 엔진 지원 및 Windows 환경 호환성을 개선하고 자동 감시(Watch) 모드를 추가했습니다.

## 0.6.3 - 2026-05-24

- 오늘의 사건 종류를 4종에서 7종으로 확장하고 보상 XP 밸런스를 조정했습니다.
- 가상 참가자의 투표 로직을 실제 역할 정보 대신 의심도(Suspicion) 기반으로 더 정교하게 튜닝했습니다.
- 결과 카드에 누가 누구에게 투표했는지 보여주는 '투표 요약' 섹션을 추가했습니다.

## 0.6.2 - 2026-05-23

- 가상 참가자의 투표 선택 이유를 진행 로그에 남겨 의사결정 근거를 확인할 수 있게 했습니다.

## 0.6.1 - 2026-05-22

- 게임 시작 전 화면에 명령 예시, 버튼 선택지, 축약 결과 카드 형식을 보여주는 채팅 응답 미리보기 패널을 추가했습니다.
- 날짜별 브리핑과 하루 1회 보너스 XP를 제공하는 오늘의 사건 모드를 추가했습니다.
- 결과 카드에 팀 결과, 획득 XP, 레벨 진행도, 칭호 변화, 주요 사건 요약을 추가했습니다.
- 가상 참가자의 랜덤 이름을 실제 사람 이름 대신 익명 닉네임형 이름으로 변경했습니다.

## 0.6.0 - 2026-05-21

- XP, 레벨, 칭호, 최근 플레이 기록을 브라우저 `localStorage`에 저장하도록 변경했습니다.
- 새 게임이나 게임 리셋 후에도 누적 XP와 레벨 진행도가 유지됩니다.
- 사이드바에 누적 XP, 현재 칭호, 최근 플레이 기록 요약을 표시했습니다.

## 0.5.0 - 2026-05-18

- Docusaurus 기반 문서 사이트를 `docs-site/`로 분리했습니다.
- 루트 스크립트에 `docs:dev`, `docs:build`, `verify:all`을 추가했습니다.
- 게임 화면의 Docs 링크를 별도 문서 사이트로 연결했습니다.
- 템플릿 샘플 문서, 블로그, 기본 이미지를 제거했습니다.

## 0.4.0 - 2026-05-18

- 낮 심문 기능을 추가했습니다.
- 역할별 카드 이미지를 추가했습니다.
- 진행 로그에 새 메시지가 생기면 자동으로 아래로 스크롤되도록 했습니다.
- 결과 카드에 XP, 레벨, 칭호 정보를 추가했습니다.

## 0.3.0 - 2026-05-18

- 혼자 플레이하는 가상 참가자 구조를 추가했습니다.
- 내 역할만 공개하고, 다른 역할은 게임 종료 후 공개하도록 바꿨습니다.
- 가상 참가자의 자동 발언과 투표 로직을 추가했습니다.

## 0.2.0 - 2026-05-18

- 라운드 진행, 밤 행동, 낮 토론, 투표, 승패 판정을 구현했습니다.

## 0.1.0 - 2026-05-18

- Next.js 기반 초기 프로젝트를 만들었습니다.
- 자동 커밋 스크립트와 기본 프로젝트 문서를 추가했습니다.
