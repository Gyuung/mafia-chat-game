# 변경 이력

## 1.1.0 - 2026-05-27

- **플레이어 정체성 강화**:
  - 이제 플레이어의 이름이 '나' 대신 봇들과 동일한 형식의 랜덤 이름(예: '회색 후드')으로 자동 할당됩니다.
  - 할당된 이름을 통해 NPC들이 플레이어를 지목하여 심문하거나 대화에서 언급하는 등 상호작용이 더욱 자연스러워졌습니다.
- **인게임 튜토리얼 시스템**:
  - 신규 사용자를 위한 5단계 인터랙티브 가이드를 추가했습니다.
  - 심문 방법, 밤 행동 선택, 투표 규칙 등을 실제 화면 구성과 연동하여 학습할 수 있습니다.
- **수사 경력 통계(Career Stats)**:
  - 총 플레이 횟수, 승률, 역할별 수행 빈도(마피아, 의사 등), 평균 생존 라운드 등 누적 데이터를 한눈에 볼 수 있는 대시보드를 구현했습니다.
- **모바일 UX 최적화**:
  - 상단바 상시 노출 및 접이식 사이드바를 적용하여 작은 화면에서도 쾌적하게 플레이할 수 있습니다.

## 1.0.0 - 2026-05-25

- **정식 버전 출시**:
  - **성격 시스템 완성**: NPC들이 성향(논리, 공격, 소심, 감성, 냉소)에 따라 대화 패턴과 투표 성향이 달라집니다.
  - **마피아 추리 단서**: 마피아 봇이 거짓말을 할 때 나타나는 언어적 습관(Tells) 시스템을 강화했습니다.
  - **결과 카드 상세화**: 맞춘 투표 수, 심문 성공 횟수, 특수 역할 액션 성공 여부 등 세부 지표를 추가했습니다.
  - **공유 기능**: 오늘의 수사 결과를 텍스트 형식으로 클립보드에 복사하여 친구들에게 공유할 수 있습니다.
  - **시각 연출**: 참가자 탈락 시 화면 흔들림 효과와 피격 효과를 추가했습니다.

## 0.9.0 - 2026-05-24

- **지능형 알고리즘 도입**:
  - 가상 참가자들이 서로의 발언과 투표 이력을 기억하여 '신뢰도'를 계산하고, 이를 기반으로 투표 타겟을 결정합니다.
  - NPC가 투표 시 해당 타겟을 선택한 '이유'를 사회자 메시지로 함께 출력하여 개연성을 높였습니다.
- **고난도 사건**:
  - 보상이 더 높은 'HARD' 난이도의 일일 사건 시나리오들을 추가했습니다.

## 0.8.0 - 2026-05-22

- **오늘의 사건(Daily Case) 모드**:
  - 매일 자정에 갱신되는 특별 시나리오 모드를 추가했습니다.
  - 하루 한 번 성공 시 추가 XP를 획득할 수 있습니다.
- **응답 미리보기**:
  - 관리자 및 개발자를 위해 채팅 플랫폼(Discord 등)에서 메시지가 어떻게 보일지 미리 볼 수 있는 패널을 구현했습니다.

## 0.7.0 - 2026-05-21

- **영속성 데이터 시스템**:
  - 플레이어의 XP, 레벨, 칭호, 최근 10판의 플레이 기록이 브라우저에 저장됩니다.
  - 레벨업 시 새로운 칭호를 획득할 수 있는 기본 시스템을 구축했습니다.

## 0.2.0 - 2026-05-18

- **핵심 게임 루프 완성**:
  - 밤(역할 액션) -> 낮(토론/심문) -> 투표 -> 승패 판정으로 이어지는 마피아 게임의 핵심 로직을 구현했습니다.
  - 가상 참가자들의 자동 대화 로직 및 랜덤 투표 기능을 추가했습니다.

## 0.1.0 - 2026-05-18

- **프로젝트 초기화**:
  - Next.js 16 및 React 19 기반의 마피아 채팅 게임 프로젝트를 생성했습니다.
  - Docusaurus 기반의 공식 문서 사이트를 구축했습니다.
