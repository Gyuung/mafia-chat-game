# 변경 이력

## 0.6.1 - 2026-05-22

- 결과 카드에 팀 결과, 획득 XP, 레벨 진행도, 칭호 변화, 주요 사건 요약을 추가했습니다.
- 가상 참가자의 랜덤 이름을 실제 사람 이름 대신 익명 닉네임형 이름으로 변경했습니다.

## 0.6.0 - 2026-05-21

- XP, 레벨, 칭호, 최근 플레이 기록을 브라우저 `localStorage`에 저장하도록 변경했습니다.
- 새 게임이나 게임 리셋 후에도 누적 XP와 레벨 진행도가 유지됩니다.
- 사이드바에 누적 XP, 현재 칭호, 최근 플레이 기록 요약을 표시했습니다.

## 0.5.0 - 2026-05-18

- Docusaurus 기반 문서 사이트를 `docs-site/`로 분리했습니다.
- 루트 스크립트에 `docs:dev`, `docs:build`, `verify:all`을 추가했습니다.
- 게임 화면의 Docs 링크를 별도 문서 사이트로 연결했습니다.
- 템플릿 샘플 문서, 블로그, 기본 이미지를 제거했습니다.

## 0.4.0 - 2026-05-18

- 낮 심문 기능을 추가했습니다.
- 역할별 카드 이미지를 추가했습니다.
- 진행 로그에 새 메시지가 생기면 자동으로 아래로 스크롤되도록 했습니다.
- 결과 카드에 XP, 레벨, 칭호 정보를 추가했습니다.

## 0.3.0 - 2026-05-18

- 혼자 플레이하는 가상 참가자 구조를 추가했습니다.
- 내 역할만 공개하고, 다른 역할은 게임 종료 후 공개하도록 바꿨습니다.
- 가상 참가자의 자동 발언과 투표 로직을 추가했습니다.

## 0.2.0 - 2026-05-18

- 라운드 진행, 밤 행동, 낮 토론, 투표, 승패 판정을 구현했습니다.

## 0.1.0 - 2026-05-18

- Next.js 기반 초기 프로젝트를 만들었습니다.
- 자동 커밋 스크립트와 기본 프로젝트 문서를 추가했습니다.
